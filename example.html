Tablemate v0.0.1

Tablemate is an awesome jQuery plugin to slide your table rows! (More table type support coming soon)

slideToggle table row.

Data 1 Data 2
Detail Data 1 Detail Data 2

slideDown table row.

Data 1 Data 2
Detail Data 1 Detail Data 2

slideUp table row.

Data 1 Data 2
Detail Data 1 Detail Data 2

Example usage:

HTML

<a href="#" class="toggle">toggle</a> <table> <tr class="row"> <td>Data 1 </td> <td>Data 2 </td> </tr> <tr class="row-details"> <td>Detail Data 1 </td> <td>Detail Data 2 </td> </tr> </table>

jQuery

$('.toggle').click(function() { $('.row-details').tableToggle(); });
